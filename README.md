# Portfolio iOS Style - Modern iPhone UI

A stunning, pixel-perfect recreation of the iOS interface built with React, TypeScript, and Tailwind CSS. This project showcases an authentic iPhone experience with a beautiful gradient wallpaper, functional lock screen, and polished home screen interface.

## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/anshul-chauhan2004/Portfolio_IOS_Style.git
cd Portfolio_IOS_Style
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:3000`
